pyherc.ai.pathfinding.a_star(start, goal, a_map)¶

start = the start node in a_map goal = the goal node in a_map a_map = a object that should inherit the Map class

returns a tuple (path, connections, uptated) where:

path is the optimal path (as a list of points) from the start to the goal. empty if not found,

connections and updated are for debugging (remove them from code if too slow..,):
connections is the came_from dictionary and uptated is the set of the connections, which were uptated from the heap
pyherc.ai.pathfinding.reconstruct_path(came_from, current_node)¶

came_from : dictuonary of nodes and nodes before them in the shortest found path current_node : the last node of the path
